COMMON ISSUES
-------------

Issue: Port 8080 not listening
Fix:   python godot_editor_server.py --port 8090
       (or launch Godot with HTTP API enabled)

Issue: Port 8082 legacy references
Fix:   Review files in warning details, update to port 8080
       (references in .bak files and docs are acceptable)

Issue: Autoload missing
Fix:   Check project.godot [autoload] section
       Ensure autoload definitions are not commented out

Issue: Script file not found
Fix:   Verify file exists at specified path
       Check for typos or moved files


REPORT FORMATS
--------------

JSON Report:
  - Machine-readable
  - CI/CD integration friendly
  - Includes all check details with timestamps
  - Structure: {timestamp, total_checks, passed, failed, warnings, skipped, checks[]}

HTML Report:
  - Human-readable dashboard
  - Color-coded summary statistics
  - Expandable check details
  - Professional styling
  - View in any modern browser


OPTIONS
-------
--json-report PATH      Generate JSON report
--html-report PATH      Generate HTML report
--timeout SECONDS       HTTP request timeout (default: 10)
--no-color              Disable colored output (for CI/CD)
--verbose               Show detailed check information
--skip-http             Skip HTTP API tests (offline mode)
--project-root PATH     Custom project root directory
--help                  Show full help message
