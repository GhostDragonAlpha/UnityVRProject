================================================================================
TROUBLESHOOTING
================================================================================

Cannot connect to API
  - Ensure Godot running with debug flags
  - Verify GUI mode (not headless)
  - Wait 5-10 seconds after launch
  - Check firewall allows port 8080

Position not changing
  - Player must be in valid scene
  - Walking controller must be activated
  - Physics engine must be running
  - Check Input system

Slow movement
  - Physics may be running slower than 90 FPS
  - Check CPU load
  - Review physics timestep

High Y-variance
  - Ground collision issue
  - Check raycast detection
  - Review physics margin

Tests timeout
  - Godot unresponsive
  - Check network connection
  - Monitor telemetry
  - Increase timeout if needed

EXTENSIONS
================================================================================

Add custom test:
  1. Create: def test_custom_name(self) -> None:
  2. Add to run_all_tests(): if not self.is_timeout(): self.test_custom_name()
  3. Create TestResult with findings
  4. Append to self.results

Customize thresholds:
  Edit validation values in each test method
  Example: if moved_distance > 0.05 (change value)

Customize output:
  Modify TestResult.format_output() method

Custom API:
  Edit __init__() parameters for host, port, timeout
