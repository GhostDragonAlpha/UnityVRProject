## Documentation Maintenance

**Adding New Documentation:**
1. Create the document in the appropriate subdirectory
2. Add a link to this index in the relevant section
3. Update the "Last Updated" date at the top of this file
4. If replacing existing docs, move old docs to `archive/obsolete/`

**Deprecating Documentation:**
1. Move the file to `docs/archive/obsolete/`
2. Remove the link from this index
3. Add a note to the old file location explaining where to find current docs
