**For ZapSplat:**
1. Visit the link

4. Remember attribution requirement (see Licensing below)

### Step 3: Convert to OGG Vorbis (if needed)

**Using ffmpeg (command line):**
```bash
ffmpeg -i input.wav -c:a libvorbis -q:a 5 output.ogg
```

**Using Audacity (GUI):**
1. Open WAV file in Audacity
2. File > Export > Export as OGG
3. Quality: 5 (good balance of quality/size)
4. Sample rate: 44100 Hz

## Troubleshooting

### "Audio file not found" errors in Godot
- Check file paths match exactly (case-sensitive)
- Verify OGG Vorbis format (not MP3 or WAV)
- Reimport: Right-click file > Reimport

### Looping sounds have clicks/pops
- Open in Audacity
- Ensure loop points match (start/end at zero crossing)
- Add tiny fade-in (0.01s) and fade-out (0.01s)

### Sounds too loud/quiet
- Adjust in Audacity before export
- Or adjust volume_db in MoonAudioManager.gd
- Game audio: -10 to 0 dB range is typical

### Can't convert to OGG
- Install ffmpeg: https://ffmpeg.org/download.html
- Or use Audacity (free): https://www.audacityteam.org/
