================================================================================
TROUBLESHOOTING
================================================================================

Common Issues and Solutions:

Issue: "Addon not found: my-addon"
  Cause: Specified addon directory does not exist
  Solution: Check addon name matches directory in addons/ folder

Issue: "Missing plugin.cfg in addon-name"
  Cause: Addon incomplete or corrupt
  Solution:
    - Add plugin.cfg from addon source, OR
    - Remove the addon directory
    - Run script again to verify

Issue: "Python not found"
  Cause: Python not installed or not in PATH
  Solution: Install Python 3.7+ or add Python directory to PATH

Issue: Permission denied error
  Cause: No write access to addons directory
  Solution: Run with administrator privileges or check permissions

Issue: Nothing happens (no output)
  Cause: Script ran silently (unlikely)
  Solution: Add -v flag for verbose output

Issue: Addon flattening failed
  Cause: Backup and restore process encountered issue
  Solution: Check for addon-name_backup_<id> directory (recovery backup)
